# Dog Breed Classification using CNN

A Deep Learning project that identifies dog breeds from images using Convolutional Neural Networks (CNN) and Transfer Learning.

## Project Overview

This application can:

- Detect whether the image contains a dog or human
- Predict the dog breed from an uploaded dog image
- If a human image is uploaded, suggest the resembling dog breed

This project uses computer vision and deep learning techniques for image classification.

## Features

- Human face detection using OpenCV Haar Cascade
- Dog detection using pre-trained VGG16 model
- Dog breed classification using CNN
- Transfer Learning using ResNet50
- Achieved **86% test accuracy**
